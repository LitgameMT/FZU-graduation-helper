# FZU毕业设计归档助手

因为教务处的归档要求写的太复杂了（也就是文字太长懒得看），于是用Claude写了个文件归档助手。
基于纯前端技术的毕业设计材料电子归档工具，依据《福州大学本科生毕业设计（论文）工作管理办法》开发。辅助文件核对（并支持自己配置ai辅助检验）与合并，支持"最终打印版"和"电子提交版"两种归档模式。

## 功能概览

### 规范依据

《福州大学本科生毕业设计（论文）工作管理办法》2019年发。

### 两种归档模式

| 模式 | 用途 | 输出 |
|------|------|------|
| **最终打印版** | 生成资料册（胶装用）+ 单独打印文件 | 资料册 PDF + 独立文件 PDF + 图纸文件夹 |
| **电子提交版** | 按学校要求生成 5 个交付物 | 任务书+开题+文献综述+外文、评审表+决议书、指导过程、查重报告 等合并 PDF |

### 核心功能
- **文件智能分类**：按文件名关键词自动匹配到对应的归档类别
- **AI 辅助识别**（可选）：接入 OpenAI 兼容 API 后，用 AI 语义匹配文件名，提高分类准确率
- **PDF 合并**：按学校规范的顺序自动合并多个 PDF
- **查重报告处理**：自动提取 AIGC 查重和传统查重报告的第一页，合并为两页 PDF
- **Word/图片转 PDF**：自动将 .docx 和图片文件转换为 PDF（支持中文）
- **文件完整性检查**：对照必选项清单，标记已上传和缺失文件
- **手动核实**：单独打印文件支持手动勾选确认（无需上传）
- **ZIP 打包下载**：所有文件按规范命名打包

### 内置资源

- **资料目录.pdf** 已内置在页面中，无需上传

## 操作流程

### 最终打印版

1. 填写学号、姓名
2. 上传资料文件（资料封面、任务书、开题报告、文献综述、外文原文及译文、中期检查表等）
3. 检查文件完整性清单：
   - 资料册（7 项胶装）必须全部上传
   - 单独打印文件（6 项）可上传或手动勾选核实
4. 点击「生成归档文件」→ 下载 ZIP

> 📘 **第一本（论文正文）**：打印所需论文符合格式要求即可，无需在本工具中处理。

### 电子提交版

1. 切换到「电子提交版」选项卡
2. 上传所有需要合并的文件
3. 勾选需要生成的文件（每个交付物可独立开关）
4. 点击「生成归档文件」→ 下载 `学号+姓名.zip`

> 📘 **文件6（毕业设计论文）**：符合格式的论文 PDF 即可，由学生自行准备。

## 文件命名规范

文件只需**包含关键词**即可被自动识别，不要求精确匹配。推荐命名示例：

| 文件 | 关键词 | 建议命名 |
|------|--------|----------|
| 资料封面 | `资料封面` | `资料封面.pdf`（**仅接受 PDF**） |
| 任务书 | `任务书` | `任务书.pdf` |
| 开题报告 | `开题报告` | `开题报告.pdf` |
| 文献综述 | `文献综述` | `文献综述.pdf` |
| 外文原文及译文 | `外文` / `原文` / `译文` | `外文原文及译文.pdf` |
| 中期检查表 | `中期检查` | `中期检查表.pdf` |
| 指导教师评审表 | `指导教师评审` | `指导教师评审表.pdf` |
| 评阅教师评审表 | `评阅教师评审` | `评阅教师评审表.pdf` |
| 决议书 | `决议书` | `决议书.pdf` |
| AIGC 查重报告 | `AIGC` | `AIGC查重报告.pdf` |
| 传统查重报告 | `查重` | `查重报告.pdf` |
| 论文检查表 | `检查表` | `检查表.pdf` |
| 图纸 | `图纸` / `A3` | `图纸-01-主电路图.pdf` |

> ⚠️ **资料封面仅接受 PDF 格式**，为防止排版错误，含有封面的文件请先转为 PDF 再导入。
> 
> ⚠️ 旧版 `.doc` 格式无法自动转换，请用 Word 另存为 `.docx` 后再上传。

## AI 辅助配置（可选）

支持接入 OpenAI 兼容 API（OpenAI / DeepSeek / 通义千问 / Kimi 等）：

1. 展开「AI 辅助识别配置」面板
2. 填写 API 地址、API Key、模型名称
3. 点击「测试连接」确认可用
4. 点击「AI 智能识别文件名」进行智能分类

API Key 仅存储在浏览器内存中，关闭页面后清除。

## 重要提醒

1. 请确保在上传文件之前，已完成**毕业论文（设计）检查表**中的所有自查项目并签字确认
2. 检查表需经指导教师审核后方可归档
3. **决议书、评阅人评审表、指导教师评审表、查重报告、论文检查表**需单独打印，**不要胶装进资料册**
4. 资料册的封面请使用论文封面模板，在标题后面加"（资料）"字样
5. 如果需要对评议表、查重报告进行合并处理，可切换到「电子提交版」模式

## 技术栈

- **PDF 处理**：PDF.js（文本提取/页面渲染）+ pdf-lib（创建/合并/拆分）
- **Word 处理**：Mammoth.js（.docx 文本提取）
- **打包下载**：JSZip + FileSaver.js
- **AI 集成**：OpenAI 兼容 API（可选）
- **纯前端**：无需后端，所有处理在浏览器中完成

## 浏览器兼容性

支持所有现代浏览器（Chrome / Edge / Firefox / Safari）。建议使用最新版本以获得最佳体验。
